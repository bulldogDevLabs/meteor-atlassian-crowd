meteor-atlassian-crowd

A Meteor package for the Atlassian Crowd nodejs module.

Status

This is v0.0.1 of the meteor package. We are just now begining to test.

Setup

  • Clone this repo
  • cd /path/to/meteor/app
  • mkdir packages
  • ln -s /path/to/meteor-atlassian-crowd packages/meteor-atlassian-crowd
  • meteor add elosoman:meteor-atlassian-crowd
  • Create an instance of crowd with your config options.
  • var atlassianCrowdConfig = { }; crowd = atlassianCrowd( atlassianCrowdConfg );

Future releases may be published through meteor atmospherejs ... stay tuned.

Documentation

Access the module in js as AtlassianCrowd.

You can create an instance of AtlassianCrowd by passing it an options doc.

var options = {
  "crowd": {
    "base": "http://crowd.yourdomain.com:8059/crowd/", 
  },
  "application": {
    "name": "username",
    "password": "password"
  }
}

crowd = new AtlassianCrowd(options);
